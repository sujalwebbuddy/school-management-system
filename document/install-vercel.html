Install on Vercel

Prerequisite: Make sure you've completed the How to push to GitHub before proceeding with How to install on Vercel.
1. Ensure your project is pushed to GitHub (see How to push to GitHub guide).
2. In Vercel, click New Project → Import your repository.
3. Framework preset: Create React App or Other. Build command: npm run build and output directory: build.
4. Add all environment variables from the Environment Variables page (REACT_APP_API_URL, etc.).
5. Deploy. After deploy, update REACT_APP_API_URL with your backend API URL.

Notes

  • This deploys only the React frontend. Backend must be deployed separately (AWS EC2, Heroku, etc.).
  • Configure REACT_APP_API_URL to point to your backend server URL.
  • MongoDB connection is configured on the backend server, not in Vercel.
  • Backend API routes are served from your Node.js server, not from Vercel.
AI/ML Requirements: SchoolHub Platform includes advanced AI features using TensorFlow.js. Ensure your backend hosting provider supports Node.js 16+ and has sufficient resources for AI processing. AWS EC2 or similar providers are recommended for optimal AI/ML performance.
This is a MERN stack application with AI/ML capabilities. The frontend (React) is deployed on Vercel, while the backend (Node.js/Express) must be deployed separately on a Node.js hosting service like AWS EC2, Heroku, or DigitalOcean. The backend requires additional system dependencies for TensorFlow.js and AI processing.
