User Guide

Complete User Guide: Learn how to use SchoolHub Platform v1.0, from initial setup to managing users, classes, homework, and communication across all user roles (Admin, Teacher, Student).

Getting Started

1. Platform Overview

SchoolHub Platform is a comprehensive enterprise Learning Management System (LMS) with multi-tenant architecture supporting multiple educational institutions. The platform includes three primary user roles with extensive features:

  • Administrator: Full system access - manage organizations, users, classes, subjects, approve registrations, subscription management, and system analytics
  • Teacher: Comprehensive teaching tools including homework creation, exam management, attendance tracking, grading, real-time chat, and performance analytics
  • Student: Complete learning environment with homework submission, exam taking, attendance viewing, chat communication, and progress tracking

2. Organization & User Registration Process

Organization Registration (For New Schools/Institutions)

If you're setting up SchoolHub for a new educational institution:

  1. Visit Homepage: Go to the SchoolHub Platform homepage
  2. Click "Register Organization": Choose the organization registration option
  3. Select Subscription Tier: Choose Primary ($29/month), High School ($79/month), or University ($199/month) plan
  4. Organization Details: Provide school name, domain, and administrator contact information
  5. Payment Setup: Complete Stripe payment setup for your chosen subscription
  6. Account Activation: Receive organization credentials and setup instructions
  7. First Admin Login: Log in with provided admin credentials to complete organization setup
Organization Setup: Each organization gets its own isolated environment with custom domain, user limits, and feature access based on the subscription tier. The first user created is automatically assigned as the organization administrator.

User Registration (For Existing Organizations)

If you're joining an existing SchoolHub organization:

  1. Visit Homepage: Go to your organization's SchoolHub login page
  2. Click "Register": Select the user registration option
  3. Organization Domain: Use your organization's specific domain or select from available organizations
  4. Complete Registration: Fill in your personal details (name, email, role, phone number)
  5. Select Role: Choose your role (Teacher or Student) within the organization
  6. Wait for Approval: Organization admin must approve your account before you can access the platform
  7. Receive Credentials: Admin will provide your login credentials via email once approved
  8. First Login: Use provided credentials to log in and complete your profile setup
Approval Required: All user registrations require administrator approval to ensure secure and controlled access. Contact your organization's administrator if your account hasn't been approved within 24 hours.
Important: All user registrations require admin approval. Contact your school administrator if your account hasn't been approved yet.
